# Cloud Chat App

## About

This is a simple chat app I built using HTML, CSS, and JavaScript. It’s just a basic project to understand how a chat interface works.

## Files

* index.html → main page
* style.css → styling
* app.js → functionality

## How to run

1. Download or clone the project

   ```
   git clone https://github.com/Tharun-GK/cloud-chat-app.git
   ```
2. Open the folder
3. Open `index.html` in your browser
